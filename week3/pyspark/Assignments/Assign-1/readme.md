# 🚀 PySpark DataFrame Operations

A beginner-friendly PySpark project demonstrating common DataFrame operations using an Employee dataset.

---

## 📌 Covered Concepts

### ✅ SELECT
- Selecting single and multiple columns
- Selecting columns with conditions

### ✅ ALIAS
- Renaming columns temporarily

### ✅ FILTER / WHERE
- Conditional filtering
- Multiple conditions using AND & OR
- String-based filtering

### ✅ WITHCOLUMNRENAMED
- Renaming DataFrame columns

### ✅ WITHCOLUMN
- Creating calculated columns
- Conditional columns
- Derived columns

### ✅ TYPECASTING
- Converting datatypes

### ✅ SORT / ORDERBY
- Sorting records

### ✅ LIMIT
- Fetching limited rows
