# PySpark Data Engineering Projects

## Overview

This repository contains three hands-on PySpark projects designed to strengthen core Data Engineering and Big Data processing skills. The projects cover data cleaning, SQL-to-PySpark transformations, analytics, window functions, ETL workflows, and business reporting using Apache Spark.

These projects were developed using **Python, PySpark, Spark SQL, and Google Colab** to simulate real-world data engineering scenarios.

---

## Tech Stack

- Python
- PySpark
- Apache Spark
- Spark SQL
- Google Colab
- Pandas (for dataset preparation)

# Project 1 – Employee Salary Analytics

### Objective

Perform employee salary analysis using PySpark DataFrames and Spark SQL while learning aggregation, filtering, window functions, and business reporting.

### Features

- Data Loading
- Data Cleaning
- Highest Paid Employees by Department
- Average Salary by City
- Salary Band Classification
- Year-wise Hiring Analysis
- Employees Earning Above Department Average
- Window Functions
- Spark SQL Queries

### PySpark Concepts

- DataFrames
- GroupBy
- Aggregations
- Window Functions
- Conditional Columns
- Joins
- Date Functions
