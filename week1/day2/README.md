# SQL Practice – Day 2

## Overview
This repository contains Day 2 SQL practice queries using MySQL.  
The exercises focus on learning different SQL clauses and operators with practical examples using an `Employees` table.

---

## Topics Covered

- SELECT Statements
- WHERE Clause
- GROUP BY
- HAVING Clause
- ORDER BY with LIMIT
- DISTINCT Keyword
- Comparison Operators
- Logical Operators
- IN and NOT IN
- BETWEEN Operator
- LIKE Operator
