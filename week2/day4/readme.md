## 🎯 Learning Objectives
- Understanding and applying SQL string functions  
- Cleaning and formatting text data  
- Extracting useful information from string columns  

---

## 🧠 Concepts Learned

- Measuring string length using `LENGTH` and `CHAR_LENGTH`  
- Converting text case using `UPPER` and `LOWER`  
- Removing unwanted spaces using `TRIM`, `LTRIM`, `RTRIM`  
- Combining strings using `CONCAT` and `CONCAT_WS`  
- Extracting substrings using `SUBSTRING`, `LEFT`, `RIGHT`  
- Finding positions of characters using `INSTR` and `LOCATE`  
- Replacing text using `REPLACE`  
- Reversing strings using `REVERSE`  
- Padding strings using `LPAD` and `RPAD`  
- Handling NULL values using `IFNULL` and `COALESCE`  
- Searching values in lists using `FIND_IN_SET`  

---

## 🚀 Outcome
- Improved ability to clean and manipulate string data in SQL  
- Better understanding of real-world data transformation techniques  
- Confidence in writing optimized queries for text processing
